== Description ==

WPEC Related Products for WP e-Commerce uses information available within the Single Product template to display related Products that belong to the same Product Category.

WPEC Related Products is a WordPress Plugin to extend the WP e-Commerce Plugin by displaying related Products to site visitors, this provides a cross-selling opportunity to other Products available from your store. It automatically selects related Products based on the Category that the currently displayed Product belongs to.
